## Problema

43 - Qual é o valor da variável `x` após as seguintes instruções.

```cs
double x = 11.09;
AddOne(x);
```

### Soluções 

```cs
x = 11.09
```
O valor de `x` fica igual (11.09) pois este (valor) não é passado como
referência.
