## Problema

8 - Indica o que é mostrado no ecrã pelo seguinte código C#. Justifica a tua
resposta.

```cs
double x = 3.1223;
Console.WriteLine(@"\\{x}\\:" + x);
```

## Soluções

### Solução 1

Como a string está em Verbatim, mas não está interpolada irá aparecer `\, {`
mas não o valor da variável.

Impresso no ecrã:
`\\{x}\\:3.1223`
