## Problema

39 - Qual o comando para eliminares uma _tag_ com etiqueta `v987`? Qual o
comando para eliminares esta _tag_ do repositório remoto `remoto-do-ze`?

### Soluções 1

Comando para eliminar uma _tag_ com etiqueta `v987` :

`git tag -d v987`

Comando para eliminar esta _tag_ do repositório remoto `remoto-do-zé` :

`git push remoto-do-zé :v987`
