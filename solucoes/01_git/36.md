## Problema

36 - Qual o comando para adicionares uma _tag_ anotada com etiqueta `v1.2` e
mensagem `Fix 10 bugs since last version` ao último _commit_ efetuado? Qual o
comando para fazeres _push_ única e exclusivamente desta _tag_ para o
repositório remoto `origin`?

### Soluções 1

Comando para adicionar uma _tag_ anotada com etiqueta `v1.2` e
mensagem `Fix 10 bugs since last version` ao último _commit_ efetuado :

´git add -a v1.2 -m Fix 10 bugs since last version`

Comando para fazer _push_ única e exclusivamente desta _tag_ para o
repositório remoto `origin` :

`git push origin v1.2`
