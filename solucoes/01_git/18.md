18 - Apagaste o ramo new-feature do teu repositório local, ramo esse que estava também mapeado (com o mesmo nome) no repositório remoto upstream. Qual é o comando para apagar o ramo também no repositório remoto?

## Soluções

### Solução 1

O comando para apagar o ramo no repositório remoto é:

```
$ git push upstream --delete new-feature
```

*Solução proposta por [André Santos](https://github.com/Snigy24)*
